# Contributing

Thank you for your interest in my personal GitHub Profile repository!

Since this repository serves as my personal portfolio and developer dashboard, **I generally do not accept Pull Requests that alter the personal content of the `README.md`** (such as changing the bio, projects, or personal links).

However, you are more than welcome to:
- **Fork this repository** and adapt the architecture, GitHub Actions, and Python scripts for your own profile (under the MIT License).
- **Report bugs** or submit PRs if you find a technical issue with the Python automation scripts (`scripts/`) or GitHub Actions workflows (`.github/workflows/`).
- **Fix typos** in the general documentation.

## Submitting a Pull Request
If you are submitting a technical fix for the automation scripts:
1. Ensure you have tested the changes locally.
2. Use the provided Pull Request template.
3. Clearly explain what the bug was and how your code fixes it.
